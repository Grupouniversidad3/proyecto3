# Directorio de Nomina (RRHH) - Equipo 3

Este es un programa de consola desarrollado en JavaScript (Node.js) para gestionar los trabajadores de una empresa y sacar las cuentas de la nomina quincenal. Permite registrar, ver el personal, hacer cambios en los datos, eliminar registros de forma segura y ver estadisticas de los costos.

## Estructura del Proyecto
Separamos el codigo en tres archivos para que sea ordenado:
1. `Archivo1.js`: Tiene el menu principal y se encarga de pedirle los datos al usuario.
2. `Archivo2.js`: Se encarga de revisar que los sueldos y los anos de servicio esten bien, y de sacar el gasto total.
3. `Archivo3.js`: Se encarga de que todo se guarde en el archivo `Archivo2.json` para que no se borre nada.
